# 一、简介

本仓库不定期收集、整理、编写常用优秀软件和服务的Dockerfile文件，分享成熟、稳定、好用的docker-compose.yml部署配置文件

当前已全部构建**X86**和**ARM**架构Docker镜像，可以直接访问[Docker Hub](https://hub.docker.com/u/siriling)官方镜像仓库拉取使用
